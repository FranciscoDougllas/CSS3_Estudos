Documentação: Box Model, Margin e Padding

O Box Model é um conceito fundamental do CSS que define como os elementos são renderizados na página. Cada elemento é representado como uma caixa retangular composta por quatro áreas principais:

  • Content (Conteúdo): onde fica o texto ou outros elementos filhos.
  • Padding: espaço interno entre o conteúdo e a borda.
  • Border: a borda ao redor do padding e do conteúdo.
  • Margin: espaço externo entre a borda do elemento e outros elementos.

Exemplo de Código

<div class="exemplo">Conteúdo</div>

.exemplo {
    margin: 20px;
    border: 3px solid #1976d2;
    padding: 15px;
    background: #e3f2fd;
}

Explicação das Propriedades

  • margin: Define o espaço externo do elemento. Pode ser individual (margin-top, margin-right, etc) ou abreviado.
  • border: Define a borda do elemento. Pode configurar largura, cor e estilo.
  • padding: Define o espaço interno entre o conteúdo e a borda.
  • background: Define a cor ou imagem de fundo do conteúdo e do padding.

Dicas

  • O padding e o border aumentam o tamanho total do elemento, a menos que você use box-sizing: border-box;.
  • O margin não afeta o tamanho do elemento, apenas o espaçamento externo.
  • Você pode inspecionar o Box Model de qualquer elemento usando as ferramentas de desenvolvedor do navegador.
