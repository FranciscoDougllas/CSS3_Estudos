Cores no CSS

Existem por padrao basico 3 formas de definir cores no CSS:

  • Nome da cor
  • Hexadecimal
  • RGB

Exemplo 4: Outras formas de definir cores

  • Cor pelo nome: orange
  • Cor hexadecimal: #00bfff
  • Cor RGB: rgb(128, 0, 128)
  • Cor RGBA (transparente): rgba(255, 0, 0, 0.5)
  • Cor HSL: hsl(120, 100%, 25%)
  • Cor HSLA (transparente): hsla(240, 100%, 50%, 0.3)
