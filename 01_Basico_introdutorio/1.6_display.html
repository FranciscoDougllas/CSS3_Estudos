Tipos de Display no CSS3

O display é uma das propriedades mais importantes do CSS, pois define como um elemento será exibido na página. Abaixo estão os principais tipos de display e exemplos de utilização:

1. display: block

Elementos de bloco ocupam toda a largura disponível e começam em uma nova linha.

Elemento 1 (block)
Elemento 2 (block)
        .exemplo-block {
            display: block;
        }

2. display: inline

Elementos inline ocupam apenas o espaço necessário e não iniciam uma nova linha.

Inline 1 Inline 2 Inline 3 
        .exemplo-inline {
            display: inline;
        }

3. display: inline-block

Combina características dos dois anteriores: fica na mesma linha, mas aceita largura e altura.

Inline-block 1 Inline-block 2 Inline-block 3 
        .exemplo-inlineblock {
            display: inline-block;
            width: 120px;
            height: 40px;
        }

4. display: flex

Permite criar layouts flexíveis e responsivos facilmente.

Flex 1
Flex 2
Flex 3
        .exemplo-flex {
            display: flex;
            gap: 10px;
        }

5. display: none

Remove o elemento da renderização da página (ele não ocupa espaço).

        .oculto {
            display: none;
        }

Exemplo: O elemento abaixo está oculto e não aparece na página.

Resumo dos principais valores de display:

  • block: Ocupa toda a largura, inicia em nova linha (ex: <div>, <h1>)
  • inline: Ocupa apenas o necessário, não quebra linha (ex: <span>, <a>)
  • inline-block: Fica em linha, mas aceita largura/altura
  • flex: Layout flexível para elementos filhos
  • none: Elemento não é exibido
