Exemplo de Variáveis em CSS

As variáveis em CSS permitem definir valores reutilizáveis para propriedades, facilitando a manutenção e a consistência do design.
No exemplo abaixo, altere o valor das variáveis em :root para ver o efeito em todo o site.

Como funciona?

  1. As variáveis são declaradas usando --nome-da-variavel: valor; dentro de um seletor.
  2. Para usar a variável, utilize var(--nome-da-variavel) em qualquer propriedade CSS.
  3. Alterando o valor da variável, todos os elementos que a utilizam são atualizados automaticamente.
