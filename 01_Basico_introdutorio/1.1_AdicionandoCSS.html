Em CSS Existem 3 Formas de Adiciona o CSS a uma Pagina HTML

Inline

Internal

External

Inline: O CSS é adicionado a cada elemento HTML, usando o atributo style.

Internal: O CSS é adicionado dentro da tag head, usando a tag style.

External: O CSS é adicionado em um arquivo externo, usando a tag link.
